# PACK & GO ( An Online Travel Management Web Application )

This project is created using MERN Technologies ( MongoDB , ExpressJs , ReactJs , NodeJs )


## If you want to clone this project then follow below steps : - 

### `git clone https://github.com/MohammedRizwan4/frontend-travel.git`


## Want to Assign your directory name then this command ( Optional ) : - 

### `git clone https://github.com/MohammedRizwan4/frontend-travel.git frontend-travel`

## Now go to sub directory frontend-travel

### `cd frontend-travel`

## If one more sub directory with frontend-travel there then go inside it ( if then only run this )

### `cd frontend-travel`

## When you react to inside all check that `public` , `src` , `etc` folders are there using ls command 

### `ls`

## Now run `npm i` or `npm install` as this project is instantiated by create-react-app using `npm` package

### `npm install`

## Now last to start the project run this command

### `npm start`
